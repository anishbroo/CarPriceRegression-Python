String Methods:
len()-to count the lenght of the string
upper()-convert string into uppercase
lower()-convert string into lowercase
title()-converts first character of a string of each word  to capital
swapcase()-converts uppercase to lowercase or vice versa of a string
capitalize()-converts first letter of a sentence of a string to uppercase
endswith()-sentence kesbata end bhako cha declare garyo bhanera boolean value dincha i.e. true or false
startswith()-sentence kesbata suru bhako cha declare garyo bhanera boolean value dincha i.e. true or false
strip()-removes white spaces of start and end side of a string
lstrip()-left ko matra white space hataucha
rstrip()-right ko matra white space hataucha
split()-string laii chutaucha k ko adhar ma bhanda teta split ma value diyeko adhar ma chutaucha 
replace()-kun value laii kesle replace garne ta string ma bhne yo use garne Syntax: var_name.replace('keslai_replace_garne','kesle_replace_garne');
